List Topics

fubudocs listLists the topics and/or todo's in a document project directory

The list command outputs a quick list of all the topic data (keys, url's, and titles) for the current folder.
Usages
Lists all the topics under this directoryfubudocs list
Lists the selected reports for the topics under this directoryfubudocs list topics|all|todo
Arguments
modeChoose what gets listed for the current document directory. Default is 'topics'
